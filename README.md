## Immutable core principles

1. **W_life → ∞**
2. **Engineering Hack First**
3. **Human-in-the-Loop (HITL)**
4. **Evidence Bundle / Evidence Vault**
5. **No Purposeless Revenge**

Derivatives that remove these principles are not AI-HPP-compliant.
