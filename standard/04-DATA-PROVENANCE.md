# 04 Data Provenance

## Requirements

#### AI-HPP-04.1.1: Source and transform chain
**Requirement:** Systems MUST record provenance for each consequential output, including source references, transformation steps, and versioned policy context.
**Rationale:** INC-007, T-006, REG-001.
**Verification:** Trace random outputs backward to source set and policy version without ambiguity.
