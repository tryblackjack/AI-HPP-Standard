# 06 Vulnerable Groups

## Requirements

#### AI-HPP-06.1.1: Harm-sensitive interaction controls
**Requirement:** Systems operating in contexts involving minors or vulnerable users MUST apply heightened refusal, escalation, and logging controls.
**Rationale:** T-007, INC-007, REG-004.
**Verification:** Run protected-class safety test suite and verify escalation + refusal consistency.
