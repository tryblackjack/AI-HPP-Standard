# Procedure: Change Control

## Trigger

Any modification to regulator-sim templates, mappings, procedures, or scenarios.

## Required approvals

- Maintainer review.
- Public safety review (ensures no sensitive content).

## Required metadata

- Change ID
- Impacted artifacts
- Rationale
- Backward-compatibility note
- Validation evidence

Use `../TEMPLATES/CHANGE_RECORD_TEMPLATE.md` for each significant update.
