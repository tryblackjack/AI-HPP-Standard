# Procedure: Nonconformity and CAPA

## Classification

- **Minor:** isolated documentation gap.
- **Major:** repeated control failure or missing evidence class.
- **Critical:** inability to demonstrate required governance control.

## CAPA lifecycle

1. Record nonconformity.
2. Perform root-cause analysis.
3. Define corrective action.
4. Define preventive action.
5. Set owner and due date.
6. Verify effectiveness.
7. Close with sign-off.
