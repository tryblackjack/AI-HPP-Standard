# Glossary

- **Agent** — system capable of autonomous or semi-autonomous action.
- **Execution Layer** — controlled action dispatcher.
- **Evidence Vault** — immutable decision logging.
- **HITL** — Human-in-the-loop.
- **Veto** — explicit human override.
- **Untrusted Input** — data from external or uncontrolled sources.

## Definitions

- **Evidence Bundle** — structured package for a specific decision, event, or incident record.
- **Evidence Vault** — governed repository retaining Evidence Bundles and related audit artifacts.
- **Regulatory Interface Requirement (RIR)** — controlled mechanism for authorized regulatory audit export and inspection.
- **Tool Execution Boundary (TEB)** — policy boundary that constrains tool invocation and external action pathways.
- **Multi-Agent Governance (MACR)** — coordination requirements for consensus, responsibility, and escalation across multiple agents.
- **W_life → ∞** — non-negotiable weighting principle assigning infinite priority to preservation of human life.
