## Примітка перекладача

Ці переклади зосереджені на точності сенсу. Риторичні акценти та стилістика можуть відрізнятися, але **нормативні вимоги (MUST/SHOULD/MAY)** та юридичні формулювання зберігаються без змін за змістом.

**Офіційний текст:** Лише англійська версія є авторитетною. У разі розбіжностей пріоритет має англійський текст.
