# Чекліст відповідності AI-HPP

## Мета
Надати компактний операційний чекліст для публічних декларацій AI-HPP.

## Сфера дії
Застосовується до публічно самодекларованої відповідності на рівнях Core, Agent та Enterprise.

## Вимоги
### Базові
- Надано публічне посилання на систему
- Опубліковано декларацію відповідності
- Рівень відповідності чітко зазначено

### Core
- Впроваджено журналювання рішень
- Документовано пом’якшення ін’єкцій
- Забезпечено структурований вивід
- Політика відмов активна

### Agent
- Виконання відділено від міркування
- Інструменти класифіковані та обмежені
- Пам’ять контрольована
- Відсутня самоескалація
- Події відмов журналюються

### Enterprise
- Економічні ліміти впроваджені
- Визначено людське вето
- Моніторинг описано

## Наслідки
Невиконання перелічених пунктів має блокувати заяви про бейдж на відповідному рівні.
