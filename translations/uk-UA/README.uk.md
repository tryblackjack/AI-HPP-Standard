# AI-HPP: Стандарт партнерства людина–машина (короткий опис)

AI-HPP — це стандарт для автономних і агентних AI-систем із реальними наслідками. Він задає **незмінні принципи**, правила підзвітності та вимоги до аудиту для систем, що приймають рішення у високоризикових доменах.

## Коротко про зміст

- **v2.2 (стабільна версія):** базові принципи безпеки для індивідуальних AI-систем.
- **v3.0 (чернетка):** екосистемний рівень, агентні системи, Zero Trust, модулі безпеки.
- **Evidence Vault:** обов'язкове логування критичних рішень для підзвітності.
