# Глосарій термінів (uk-UA)

Цей глосарій фіксує узгоджені переклади ключових термінів для всіх українських документів.

| English | Українська | Примітка |
|---|---|---|
| Evidence Vault | Сховище доказів (Evidence Vault) | Термін залишати в дужках при першій згадці. |
| Human-in-the-Loop (HITL) | Людина в циклі (HITL) | Абревіатура HITL зберігається. |
| Failure-First Framing | Підхід «спочатку збої» | Мова про фокус на відмовах/збоях. |
| Anti-Slop Clause | Положення Anti-Slop | Назва лишається англійською. |
| Agent identity compromise / impersonation | Компрометація ідентичності агента / підміна агента | Вживати як технічний термін. |
| Regulated Cognitive Intervention (RCI) | Регульоване когнітивне втручання (RCI) | Абревіатура RCI зберігається. |

## Нормативні модальні дієслова (RFC 2119)

- **MUST** → **ПОВИНЕН** (обов'язкова вимога)
- **SHOULD** → **СЛІД** (рекомендована, але не обов'язкова вимога)
- **MAY** → **МОЖЕ** (дозволено/опційно)

Ці відповідники використовуються **послідовно** в усіх українських перекладах.
